# react-ssr

Base project for server side rendering with React v16.

## Stack

* React
* ReactDOM
* ReactRouter
* Webpack
* Express
* Helmet

## Structure

```
src
├── client
├── server
│   ├── routes
|   └── utils
└── shared
    └── app
```

## Setup

Restore dependencies and start development server:

```bash
yarn && yarn dev
```

Then, open [localhost:3001](http://localhost:3001/)
