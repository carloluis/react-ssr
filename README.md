# react-ssr

Base project for server side rendering with react.

## Stack

* React
* ReactDOM
* ReactRouter
* Webpack
* Express

## Structure

```
src
├── client
├── server
│   ├── routes
|   └── utils
└── shared
    └── app
```

## Setup

Restore all dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Then, open [localhost:3001](http://localhost:3001/)
